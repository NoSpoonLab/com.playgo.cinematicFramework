# PlayGo Cinematic Framework

Paquete UPM de prueba para Unity desarrollado por Play&Go Experience.

Este paquete sirve como base para la creación de módulos reutilizables dentro de proyectos Unity, siguiendo una arquitectura modular basada en Unity Package Manager (UPM).
